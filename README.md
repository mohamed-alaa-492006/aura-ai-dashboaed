# Aura Next.js Admin Dashboard Template

A modern, fast, and responsive Admin Dashboard template built with **Next.js 15 (App Router)**, **Tailwind CSS**, and **Recharts**.

## Features

**Next.js App Router** structure.
Styled with **Tailwind CSS**.
Interactive Charts powered by **Recharts**.
**Users Management** with search and state modifications.
**Analytics & Reports** overview.
**Settings & Profile** management tabs.
Fully Responsive on Desktop, Tablet, and Mobile.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:
npm run dev

Open http://localhost:3000 with your browser to see the result.

src/
├── app/ # App Router Pages (Overview, Users, Analytics, Settings)
├── components/ # Reusable UI Components (Sidebar, Header, Chart, Table)
└── lib/ # Dummy Data & Helper Configs
