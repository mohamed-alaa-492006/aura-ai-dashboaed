# 🚀 Aura - Next.js Admin Dashboard Template

A modern, fast, and responsive Admin Dashboard template built with **Next.js 15 (App Router)**, **Tailwind CSS**, and **Recharts**. Designed specifically for SaaS products, modern web apps, and management systems.

---

## ✨ Features

- **Next.js 15 App Router:** Modern architecture for lightning-fast performance and SEO optimization.
- **Tailwind CSS:** Fully custom-styled components, easily customizable colors, fonts, and themes.
- **Interactive Charts:** Beautiful data visualization powered by **Recharts**.
- **Users Management:** Full UI setup with real-time search filtering, status badges, and action modals.
- **Analytics & Reports:** Comprehensive metrics overview with conversion rates, sales data, and performance stats.
- **Settings & Profile:** Pre-built profile management, notification settings, and security controls.
- **Fully Responsive:** Perfectly optimized and tested across Desktop, Tablet, and Mobile screens.

---

## 🛠️ Tech Stack

- **Framework:** Next.js 15
- **Styling:** Tailwind CSS
- **Icons:** Lucide React / React Icons
- **Charts:** Recharts
- **Language:** JavaScript / TypeScript

## 🚀 Getting Started

Follow these steps to run the project locally on your machine.

### Prerequisites

Make sure you have **Node.js 18.x** or higher installed on your computer.

### Installation

1. **Clone or extract the project folder:**
   ```bash
   cd aura-ai-dashboard
   ```
   Install dependencies:
   npm install

# or

yarn install

# or

pnpm install

Run the development server:
npm run dev

# or

yarn dev

# or

pnpm dev

Open in Browser:
Open http://localhost:3000 with your browser to see the live application.

src/
├── app/ # App Router Pages (Overview, Users, Analytics, Settings)
├── components/ # Reusable UI Components (Sidebar, Header, Chart, Table, Card)
└── lib/ # Dummy Data, Constants & Helper Functions

To build the project for production, run:

npm run build

Then, start the production server:
npm start
